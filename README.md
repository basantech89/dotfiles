# Dotfiles

## Installation

Run `make` to install dotfiles in your system

### You can also install dotfiles for just one specific tool, for example

Run `make fish` to add configurations files only for fish shell

## Deletion

Run `make delete` to remove the dotfiles from your system

### You can also remove dotfiles for just one specific tool, for example

Run `make delete_fish` to remove configurations files only for fish shell

## Details

These dotfiles adds configuration files for below tools

1. fish shell

2. zsh shell

3. neovim

4. warp terminal

5. starship
